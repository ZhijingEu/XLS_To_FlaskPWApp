Epidemic Contagion Growth Estimate - Demo App


Sorry...you need to be online to use this :'(

This is a minimal viable demo app that shows how an Excel spreadsheet tool can be converted into a web app

The tool plots contagion growth rates (i.e daily and cumulative no of cases) for a given set of parameters based on a Logistic Growth Curve and a SIR (Susceptible,Infected,Recovered) Model
